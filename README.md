# 🛡️ CySA+ Cybersecurity Lab Analysis Project

## 📌 Overview
This project demonstrates practical cybersecurity skills through a structured lab environment involving network setup, traffic analysis, vulnerability assessment, and incident response concepts.

---

## 🧪 Lab Components
- Virtual Machine setup
- GNS3 network simulation
- Cisco router configuration
- Network reconnaissance
- Wireshark packet analysis
- SSH security analysis
- Vulnerability scanning
- Intrusion Detection concepts

---

## 🔍 Key Activities

### 🌐 Network Setup
Configured virtual machines and simulated network topology using GNS3 and Cisco router configurations.

### 🔎 Reconnaissance
Performed basic network scanning and information gathering to identify potential targets.

### 📊 Traffic Analysis (Wireshark)
Analyzed captured packets including:
- DNS queries
- TCP communication
- LDAP traffic

### 🔐 TCP Flags Understanding
Explained key TCP flags:
- SYN → Connection initiation  
- ACK → Acknowledgment  
- RST → Connection termination  

### 🚨 SSH Security Analysis
Identified risks such as:
- Credential attacks
- Unauthorized access
- Phishing and system compromise  

### 🛠️ Vulnerability Scanning
Explored scanning techniques and tools such as Nessus for:
- Identifying system weaknesses  
- Assessing attack surfaces  

### ⚖️ Risk Management Strategies
- Risk Mitigation  
- Risk Avoidance  
- Risk Transference  
- Risk Acceptance
